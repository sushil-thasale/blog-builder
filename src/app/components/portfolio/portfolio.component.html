Soccer Application:
European Soccer-360

Soccer-360 is a mobile-first, interactive, SPA developed using MEAN (Angular 4) stack. It allows user to find a common platform for sharing their interests in European Soccer. Users can view live scores, search for Leagues, Teams and Players and add them to their favorites! This website also allows users to follow other users with similar interests! It also enables the users to latest news!




Blog Builder

A Website that allows Developers to create and maintain their own blogs. A user can have multiple blogs with multiple pages in each blog website. He can design the pages by selecting different widgets and reorder them to design the page according to his liking.
